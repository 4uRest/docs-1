## User: Пользователь

> Пример объекта: 

```json
{
   "id":9,
   "posPass":"1234",
   "name":"Вова",
   "deleted":0,
   "admin":true,
   "inn":"",
   "extras":{

   }
}
```

### Свойства

Свойство | Описание
-------- | --------
id | ID пользователя
posPass | PIN код входа
name | Имя пользователя
delete | Признак что пользователь удален: 1 — удален, 0 — нет
admin | Есть ли у пользователя права администратора
