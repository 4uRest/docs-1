## User: User

> Object example: 

```json
{
   "id":9,
   "posPass":"1234",
   "name":"Vladimir",
   "deleted":0,
   "admin":true,
   "inn":"",
   "extras":{

   }
}
```


### Properties

Property | Description
-------- | -----------
id | User ID
posPass | PIN code for register
name | User name
admin | The user having administrator permission
delete | The status of the user being removed: 1 — removed, 0 — not removed
